# 📄 Rotated PDF Tool

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://easyusehub.github.io/rotated-pdf-tool/) [![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Easily **rotate PDF pages online** with live previews. Fully client-side, no API required.  

---

## Features

- Upload PDF and view **live thumbnails**  
- Select single or multiple pages to rotate  
- Quick rotate buttons: 90°, 180°, 270°  
- Rotate **all pages** at once  
- Download rotated PDF instantly  
- 2 ad placeholders (Top & Middle)  
- Responsive & user-friendly  

---

## How to Use

1. Upload your PDF.  
2. Select pages to rotate.  
3. Rotate using quick buttons or dropdown.  
4. Click **Download** to save rotated PDF.
